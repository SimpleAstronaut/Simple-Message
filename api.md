## 2.api列表
###2.0 状态码信息
| 状态码 | 信息 |
| ---- | ---- |
| 200 | 请求/操作成功 |
| 300 | 请求/操作失败 |
### 2.1 首页接口
地址: `http://127.0.0.1:8081/`或`http://127.0.0.1:8081/index.html`   
调用方式：`GET`  
调用示例: 略
返回示例：略  
### 2.2 添加留言
地址：`http://127.0.0.1:8081/add`  
调用方式：`GET`
请求参数：
| key | value |
| ---- | ---- |
| user | 显示名称 |
| msg | 留言内容 |
请求示例：
```
http://127.0.0.1:8081/add?user=test&msg=test
```
返回示例：
```json
{ "status" : 200 }
```
### 2.3 获取全部留言
地址：`http://127.0.0.1:8081/get`  
调用方式：`GET`
请求参数：无
请求示例：
```
http://127.0.0.1:8081/get
```
返回参数：
| 参数 | 信息 |
| ---- | ---- |
| user | 显示名称 |
| msg | 留言内容 |
返回示例：
```json
[{"user":"test","msg":"hello"}]
```
